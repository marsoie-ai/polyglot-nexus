### *Polyglot-Nexus*                      **Architect: Ben Mahmoud**

### *Multilingual Semantic Logic Adapter for Computer Science Education*

**The Idea** A high-performance AI engine that adapts complex Computer Science concepts into several distinct cultural and linguistic frameworks. Designed for international educators, it ensures technical rigor while respecting regional pragmatic norms.

## 🌍 Supported Complex Frameworks
* **German:** Formal 'Sie' / Technical Precision
* **Maltese:** Semitic-Romantic Hybrid with full diacritic support (ċ, ġ, ħ, ż)
* **Arabic:** Modern Standard / High-Context Honorifics
* **French, Italian, Spanish:** Regional pedagogical adaptations

## 🛠️ Technical Stack
* **Model:** Llama 3.3 70B
* **Inference:** [Groq LPU™](https://groq.com/) (Ultra-low latency)
* **Language:** Python 3.10+
* **Deployment:** Streamlit (Planned)

## 📖 Pedagogy
This was developed to solve the "Linguistic Logic Gap" thus ensuring students understand *how* a concept works in their native logical structure before translating it to syntax.

**🏫 Polyglot-Nexus: Semantic Logic Adapter**

**Target:** Cambridge International Education (CIE) CS Curriculum

**🎯 Project Vision**

Polyglot-Nexus is a high-fidelity educational tool designed to bridge the gap between abstract Computer Science concepts and diverse cultural frameworks. By leveraging LLM orchestration, the engine adapts complex logic (e.g. Time Complexity, Logic Gates, etc...) across 7 linguistic and cultural lenses.

**🛠️ Engineering Highlights**

    **1) Dynamic Semantic Mapping: Orchestrated system prompts to maintain academic rigor while adjusting cognitive load for Primary through A Level pathways.**
    **2) RTL Rendering Engine: Implemented arabic_reshaper and python-bidi within an xhtml2pdf pipeline to solve complex Right-to-Left (RTL) typesetting challenges in PDF generation.**
    **3) Interactive Analytics: Developed a real-time dashboard to visualize lesson depth and linguistic framework verification.**
    **4) Prompt Engineering: Demonstrated advanced LLM control through weighted constraint logic.**
